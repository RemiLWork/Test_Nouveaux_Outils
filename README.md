#  Tests de nouveaux outils de visualisation

## Objectif
Explorer des outils alternatifs à Power BI pour enrichir mes capacités de datavisualisation et de storytelling visuel.

---

## Datawrapper

**Mon test :**
https://datawrapper.dwcdn.net/j0zjg/1/

**Points forts :**
- Interface simple, sans code
- Export facile vers GitHub ou site web

**Limites :**
- Personnalisation plus limitée que Power BI
- Peu d’interactivité avancée

---

## Observable Plot

**Points forts :**
- Visualisations dynamiques et modernes
- Intégration possible dans GitHub Pages

**Limites :**
- Nécessite un petit temps d’adaptation
- Moins accessible pour un public non technique
